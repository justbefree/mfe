# 错误2

Uncaught Error: Shared module is not available for eager consumption: 6196
1

解决：增加bootstrap.js 通过 index.js 异步加载页面 主要原因是 remote 暴露的 js 文件需要优先加载，如果 bootstrap.js 不是一个异步逻辑，在 import User 的时候，会依赖 app 的 app.js，如果直接在 index.js 执行，app 的 app.js 根本没有加载，所以会有问题
